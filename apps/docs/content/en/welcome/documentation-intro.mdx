# Documentation Overview

The docs in this project are built with **Nextra + Next.js App Router**.

If this is your first time here:

1. Read **[Getting Started](/en/getting-started)** for setup and scripts.
2. Compare templates in **[Templates](/en/templates)**.
3. Go to **[Backend Integration](/en/backend/proxying-api-requests)** if you need API setup.

## Repo Structure

- `apps/basic`: English-only admin template
- `apps/with-lingui`: Internationalized template with Lingui
- `apps/docs`: Documentation site

## Recommended Reading Order

Welcome → Getting Started → Templates → Development → Building → Testing → Deployment
