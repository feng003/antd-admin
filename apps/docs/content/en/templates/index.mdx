# Templates

This repository provides two templates:

- **[basic](/en/templates/basic)**: Minimal setup, English UI by default.
- **[with-lingui](/en/templates/with-lingui)**: Includes Lingui i18n workflow and bilingual setup.

## How to Choose

- Choose **basic** if you only need one language.
- Choose **with-lingui** if you need structured multilingual support from day one.
