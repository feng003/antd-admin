# basic

`apps/basic` is a streamlined admin template focused on fast startup and simple extension.

## Good Fit

- Internal tools
- Small-to-medium dashboard projects
- Teams not requiring i18n yet

## Highlights

- Clean route organization
- Common data table and form patterns
- Practical defaults for request handling and page layout

## Start Development

```bash
pnpm --filter ./apps/basic dev
```
