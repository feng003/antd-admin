# with-lingui

`apps/with-lingui` adds multilingual capabilities based on Lingui while keeping the same admin architecture.

## Good Fit

- Products serving multiple regions/languages
- Teams that want standardized translation workflow
- Projects requiring language switch in UI

## Highlights

- Lingui message extraction and compilation flow
- Language resource organization
- Built-in language switching patterns

## Start Development

```bash
pnpm --filter ./apps/with-lingui dev
```
