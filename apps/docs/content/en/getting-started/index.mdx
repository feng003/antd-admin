# Getting Started

## Requirements

- Node.js 20+
- pnpm 9+

## Install Dependencies

```bash
pnpm install
```

## Run Projects

```bash
# Run docs
pnpm --filter ./apps/docs dev

# Run basic template
pnpm --filter ./apps/basic dev

# Run Lingui template
pnpm --filter ./apps/with-lingui dev
```

## Build

```bash
pnpm --filter ./apps/basic build
pnpm --filter ./apps/with-lingui build
```
